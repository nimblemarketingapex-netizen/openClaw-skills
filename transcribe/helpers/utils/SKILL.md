# Utils Helpers

## Описание
Вспомогательные функции для работы с текстовыми файлами и результатами скилла.

### Функции

1. **save_text_to_file**  
   - Сохраняет текст в указанный файл на сервере
   - Используется для транскриптов, резюме и списка задач

2. **Другие возможные утилиты**  
   - Конвертация форматов файлов, временные директории и т.п.
   - Используются только внутри `main_skill_core.py` и `helpers/transcribe.py`
