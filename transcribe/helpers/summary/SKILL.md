# Summary Helpers

## Описание
Подскиллы для генерации текстов на основе транскрипта Faster‑Whisper.

### Функции

1. **short_summary**  
   - Генерирует краткое резюме совещания.
   - Используется после транскрипции.

2. **full_summary**  
   - Генерирует полное резюме с разделением спикеров (если возможно).

3. **tasks**  
   - Составляет список задач для сотрудников на основе транскрипта.
   - Экономит токены, так как использует уже готовый текст.

## Использование
- Принимает транскрипт из `main_skill_core.py`
- Возвращает текстовые результаты для дальнейшего использования в других скиллах или сохранения.
