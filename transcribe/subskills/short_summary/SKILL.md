# Short Summary

Бот использует готовый транскрипт для создания короткого резюме без повторной транскрипции.

## Когда использовать

- После завершения транскрипции аудио/видео.
- Если пользователь хочет получить краткий обзор совещания.

## Примеры запросов

- "Сделай краткое резюме файла"
- "Выведи основные моменты совещания"

## Шаги

1. Получить транскрипт из `main_skill_core.py`.
2. Сгенерировать краткое резюме.
3. Вернуть текст пользователю и/или сохранить в файл `short_summary.txt`.
