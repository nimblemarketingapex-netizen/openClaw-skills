# Tasks

Бот использует транскрипт совещания, чтобы выделить действия и задачи для команды.

## Когда использовать

- После транскрипции аудио/видео.
- Если пользователь хочет получить конкретные задачи для сотрудников.

## Примеры запросов

- "Составь список задач для команды"
- "Что нужно сделать после совещания?"

## Шаги

1. Получить транскрипт из `main_skill_core.py`.
2. Сформировать список задач.
3. Сохранить задачи в `tasks.txt` и вернуть пользователю.
