# Marketing Strategy
## When to use
Используется, когда пользователь просит:
- Полную маркетинговую стратегию для бизнеса или проекта
- Конкретные рекомендации по продвижению, контенту и рекламе
- SWOT-анализ и план действий на ближайшие месяцы

## Steps

1. **Сбор информации**
   - Определи нишу бизнеса.
   - Определи целевую аудиторию (ЦА) с детализацией: демография, боли, потребности.
   - Определи текущий оффер, позиционирование и УТП.
   - Собери данные о ключевых конкурентах: офферы, маркетинговые каналы, сильные и слабые стороны.

2. **Анализ (SWOT)**
   - Сильные стороны (Strengths)
   - Слабые стороны (Weaknesses)
   - Возможности (Opportunities)
   - Угрозы (Threats)
   - Все пункты обязательно аргументировать примерами из бизнеса и рынка.

3. **Разработка стратегии**
   - **Привлечение клиентов:** каналы, методы, этапы воронки продаж.
   - **Контент-стратегия:** типы контента, частота публикаций, платформы.
   - **Рекламная стратегия:** таргет, бюджеты, KPI, тестирование гипотез.
   - **Быстрые улучшения:** низкозатратные изменения для ускоренного эффекта.
   - **План на 3 месяца:** конкретные действия по неделям с ответственными, метриками и целями.

4. **Формат ответа**
   - Структурированный JSON:
     ```json
     {
       "business_summary": "...",
       "audience": "...",
       "offer_and_positioning": "...",
       "swot": {
         "strengths": ["..."],
         "weaknesses": ["..."],
         "opportunities": ["..."],
         "threats": ["..."]
       },
       "strategy": {
         "customer_acquisition": ["..."],
         "content_strategy": ["..."],
         "ad_strategy": ["..."],
         "quick_wins": ["..."],
         "3_month_plan": {
           "month_1": ["..."],
           "month_2": ["..."],
           "month_3": ["..."]
         }
       }
     }
     ```
   - Или Markdown-структура с заголовками:
     ```

     ...
     ## Target Audience
     ...
     ## Offer and Positioning
     ...
     ## SWOT Analysis
     ### Strengths
     - ...
     ### Weaknesses
     - ...
     ### Opportunities
     - ...
     ### Threats
     - ...
     ## Strategy
     ### Customer Acquisition
     - ...
     ### Content Strategy
     - ...
     ### Advertising Strategy
     - ...
     ### Quick Wins
     - ...
     ### 3-Month Plan
     #### Month 1
     - ...
     #### Month 2
     - ...

     - ...
     ```

## Notes
- Обязательно **конкретные действия** с шагами, чтобы пользователь мог их сразу применять.
- Можно подключить модель OpenClaw для генерации рекомендаций на основе текста или данных о бизнесе.
- Для улучшения стратегии использовать дополнительные источники: сайты конкурентов, соцсети, отзывы клиентов.
