# Auto-Answers Module

## Возможности
- получение новых комментариев/вопросов через API;
- генерация ответов через AI;
- отправка ответа на маркетплейс;
- правила и шаблоны тональности;
- логирование обработанных сообщений.

---

## Требования
- доступ к API маркетплейса;
- интернет;
- Python 3.10+;
- токен продавца.

---

## Как работает
1. скилл периодически запрашивает новые комментарии;
2. комментарий передаётся в AI;
3. AI генерирует ответ;
4. ответ отправляется через API;
5. статус фиксируется в логах.

---

## План
- MVP (базовые ответы);
- шаблоны;
- фильтры по тональности;
- интеграция с Telegram.
