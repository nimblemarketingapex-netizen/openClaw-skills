# Forecast Module

## Возможности

✔ прогноз дней до окончания товара  
✔ рекомендации по пополнению  
✔ анализ рисков дефицита  
✔ интеграция с аналитикой и складом

---

## Как работает

1. берём аналитику продаж;
2. берём остатки склада;
3. считаем средние продажи;
4. прогнозируем дни до дефицита;
5. выдаём рекомендации.

---

## Требования

- данные аналитики;
- данные склада;
- Python 3.10+;
- доступ к API (если нужен).
