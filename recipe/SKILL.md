# Recipe Skill

Навык помогает подбирать блюда:

- поиск рецептов по продуктам  
- советы по приготовлению  
- получение полного рецепта  
- подбор по настроению

---

## Что делает

Скилл:

- подсказывает, что приготовить из имеющихся продуктов  
- предлагает варианты блюд  
- показывает инструкцию и ингредиенты  
- учитывает тип блюда (быстро, уютно, праздник)

---

## Примеры использования

### Поиск по холодильнику

```python
skill.by_fridge(["chicken", "rice"])

- список подходящих рецептов
- ID рецептов для получения деталей

# Получить рецепт
skill.get_recipe("52772")

- название блюда
- категория
- регион кухни
- ингредиенты
- инструкция
- картинка (если есть)

# Подбор по настроению
skill.suggest(mood="quick")

**Mood:**

quick — быстро
party — праздник
comfort — уютно
healthy — полезно

Если точных совпадений нет — скилл предлагает похожие варианты.

**Выходные данные**

Ответ в формате JSON:

{
  "success": true,
  "data": {
    "recipes": [...]
  }
}
